# Таблица с аниме 

#### Демо https://pavepo-frontend-task.vercel.app/

### Для начала склонируйте проект
```bash
git clone https://github.com/TimurZheksimbaev/Pavepo-Frontend-Task.git
cd Pavepo-Frontend-Task
```
### Установите зависимости
```bash
yarn install
npm install
```

### Запустите проект
```bash
yarn dev
npm run dev
```
